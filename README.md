# Reaction Checker

A simple reaction time game. Test how fast you can respond when the screen turns green.

## How to Play

1. Click **Start** (or press Space)
2. Wait for the screen to turn green
3. Click (or press Space) as fast as you can
4. See your reaction time in milliseconds

## Features

- Best score and average tracking
- Scores saved to localStorage
- Works on desktop and mobile
- Installable as PWA

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
